Detailed Description

template<typename PosInd, typename PosVec>
class Fields< PosInd, PosVec >

A class for all fields.

The template class defines all fields on a mesh. Thus a constant reference to the mesh for the field is set during construction. Vector arrays for the fields are also allocated during construction.
