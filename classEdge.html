Detailed Description

template<int pdim>
class Edge< pdim >

Class describing edges.

Points to two VertexNode describing a line segement. Is also conditionally a friend of NodeElement1D when the class is Edge<1>. This is to allows NodeElement1D to read the variables in Edge directly. (This is not necessary, but I think it will be convenient.)
