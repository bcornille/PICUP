Detailed Description

template<typename PosInd, typename PosVec>
class Mesh< PosInd, PosVec >

A template class for the mesh.

The template class allows for using 1D, 2D, or 3D strucutres for mesh logical indices and coordinates independently.

Since all available dimensional types of this class must be instantiated, the one that is to be used will need to have its member vector arrays resized explicitly during runtime.

template<typename PosInd , typename PosVec >
PosVec Mesh< PosInd, PosVec >::sampleMesh ( )

Randomly sample the mesh space for a position.

A generalized sampling of the mesh volume is either not possible or not advised.

Currently this only returns a position sampled uniformly from the total volume area. In order to circumvent the costly function of finding the cell of a particle after sampling it may be advisable to sample the cells as a discrete distrubution weighted by volume, then uniformly sample the cell volume afterward. Both values can then be given simultaneously.

Returns
position of a sampled mesh volume

template<>
double Mesh< int, double >::sampleMesh ( )

Uniformly distributed sampling of the mesh region. This may be a bad way to sample the mesh because it is independent of the mesh elements so an expensive step of finding the which mesh element the sampled point occupies must be done afterward. While this doesn't pose an issue for an evenly spaced mesh, the implementation and standardization of this step may want to be reconsidered.
